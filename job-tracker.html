Bossa Nova Job Tracker

4 week project

Please check out the live site and the Github repo

Our team of 6 researched, designed and built this app from scratch in 4 weeks. This is the capstone project of my time at the School of Code.

Our tech stack included React, React Router, Tailwind CSS, React Testing Library and Cypress. I also focused on creating the RESTful API routes and setting up authentication with Auth0.

This project has helped me see the value of agile working practices like Scrum and Kanban, setting up a good CI/CD pipeline and how to resolve conflicts and overcome difficulties.
