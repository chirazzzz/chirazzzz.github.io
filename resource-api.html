Plethora Resource API

1 week project

Please check out the live site and the Github repo

Unfortunately, the site isn't functional anymore due to our teams School of Code Netlify account having some features taken away. This means you cannot search for resources anymore however you can see the design and styling work.

Our team of 3 designed and built this app from scratch in a week. This was the first major group project during my time at the School of Code.

Our tech stack included React, Node, Express and PostgreSQL. I designed and coded the Front End, helped create API routes and deployed the app onto Netlify.

I learnt a huge about connecting the backend and frontend together which was fascinating. I also practiced creating wireframes with Figma and logos with Canva.
