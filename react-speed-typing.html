SpeedTyping Game

React & React Hooks

Please check out the live site and the Github repo

This is game I built to improve my knowledge of React hooks and as a challenge to create something I'd actually like to play.

The tech stack was just React however the main aim was to practice using useState, useEffect and useRef hooks and styling the game by following mobile-first design practices.

I had a great time building this game and really wanted to recreate a retro, 8-bit style. I'm delighted with the results. My best score is 13 words!
